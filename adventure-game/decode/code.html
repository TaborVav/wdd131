DECODE

  1. Choose your mode: Use the toggle switch to select either Encode or Decode. Encoding transforms your text into a coded format, while decoding returns the text to its original form.
  2. Set the Shift Value: Enter a number (shift value) into the box. This number determines how many positions each letter in your text will be shifted.
  3. Enter Your Text: Type or paste the text you want to encode or decode into the input area.
  4. Submit: Click the Submit button to apply the shift. The result will appear in the output area.
